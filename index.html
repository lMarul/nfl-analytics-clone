Create New Application Insights Resource

Option 1: Azure Portal (Recommended)

  1. Go to Azure Portal
  2. Click "Create a resource" → Search for "Application Insights"
  3. Fill in: Resource name, Region, Resource group
  4. Click "Review + Create" → "Create"
  5. After creation, go to resource → Copy the Connection String
  6. Paste it in the form below to update this page

Option 2: Azure CLI

az monitor app-insights component create \
  --app MyApp \
  --location eastasia \
  --resource-group MyResourceGroup \
  --application-type web

Option 3: PowerShell

New-AzApplicationInsights -Name "MyApp" `
  -ResourceGroupName "MyResourceGroup" `
  -Location "eastasia" `
  -Kind web
